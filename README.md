# Trip Planner Conversational Agent 🌍
Built with Google Cloud
The Trip Planner Conversational Agent is an AI-powered assistant designed to help users plan trips through natural, engaging conversations. Leveraging Google Cloud’s conversational AI tools, the bot provides personalized support, basic travel assistance, and a smooth user experience.
# ✨Features
1. Natural Conversations
   Powered by Google Cloud Dialogflow / Vertex AI
   Understands user queries in everyday language
   Provides contextual and interactive responses
2. Basic Support
   Offers essential travel planning help (destinations, tickets and return)
   Handles FAQs and general queries
3. Personalized Bot
   Learns user preferences over time
   Provides tailored recommendations

# 🛠️ Tech Stack
1. Google Cloud Platform (GCP)
   Dialogflow CX / Vertex AI for conversational intelligence
   Cloud Functions for backend logic

# 📌 Future Enhancements
1. Integration with Google Maps API for live routes
2. Real-time flight and hotel booking support
3. Multi-language support for global travelers
